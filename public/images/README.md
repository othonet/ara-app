# Imagens do Sistema

## Imagem do AGRO

Para adicionar a imagem do AGRO na tela de login:

1. Coloque sua imagem do AGRO neste diretório com o nome: `agro-image.jpg`
2. Formatos suportados: JPG, PNG, WebP
3. Tamanho recomendado: 800x600px ou proporção similar
4. A imagem será exibida automaticamente no lado direito da tela de login

### Exemplo de comando para adicionar a imagem:

```bash
# Se você tiver a imagem em outro local
cp /caminho/para/sua/imagem.jpg public/images/agro-image.jpg

# Ou faça upload via SFTP/SCP
```

Se a imagem não for encontrada, um fallback decorativo será exibido automaticamente.
